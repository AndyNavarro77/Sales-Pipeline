# 📊 Sales Data Pipeline - End-to-End Analytics Project

## 🚀 Project Overview

This project simulates a real-world business scenario where a complete data pipeline was designed and implemented to transform raw operational data into actionable business insights.

The goal is not just to process data, but to demonstrate how a data analyst can:

- Identify business opportunities
- Improve processes
- Generate actionable insights
- Support decision-making

---

## 🎯 Business Problem

Many companies operate with fragmented data sources (Excel files, internal systems, etc.), leading to:

- Lack of visibility into performance
- Decisions based on intuition
- Operational inefficiencies
- Missed revenue opportunities

---

## 💡 Solution

A fully structured data pipeline was developed to:

1. Extract sales data from Excel files
2. Clean and transform data using Python
3. Calculate key business metrics (KPIs)
4. Store data in a relational database (MySQL)
5. Visualize insights through an interactive Power BI dashboard

---

## 🧠 Business-Oriented Approach

This project is built with a strong business mindset, not just technical execution.

Key metrics include:

- Revenue
- Units sold
- Sales performance by seller
- Channel performance
- Time-based trends

It answers questions like:

- Which products generate the most revenue?
- Which sales reps are underperforming?
- Which channels are most profitable?
- How are sales evolving over time?

---

## ⚙️ Tech Stack

- Python (ETL)
- Pandas (data processing)
- MySQL (database)
- SQL (analytics queries)
- Power BI (data visualization)
- Excel (data source)

---

## 🔄 Data Pipeline


Excel → Python (ETL) → MySQL → Power BI Dashboard


---

## 📊 Dashboard (Power BI)

Designed for business users with a focus on clarity and decision-making:

### Key KPIs:
- Total Revenue
- Total Units Sold
- Average Ticket
- Sales by Channel
- Sales Performance by Seller

### Visualizations:
- Sales trends over time (line chart)
- Sales by product (bar chart)
- Channel distribution (pie chart)
- Seller ranking
