# 📊 Pipeline de Datos de Ventas - Análisis End-to-End

## 🚀 Descripción del Proyecto

Este proyecto simula un entorno real de negocio donde se diseñó e implementó un pipeline de datos completo (end-to-end) para transformar datos operativos en información estratégica accionable.

El objetivo principal no es solo procesar datos, sino demostrar cómo un analista puede:

- Detectar oportunidades de mejora
- Optimizar procesos
- Generar insights de negocio
- Apoyar la toma de decisiones

---

## 🎯 Problema de Negocio

Muchas empresas operan con datos dispersos (Excel, sistemas internos, etc.), lo que genera:

- Falta de visibilidad sobre el rendimiento
- Decisiones basadas en intuición
- Ineficiencias operativas
- Pérdida de oportunidades de ingresos

---

## 💡 Solución

Se desarrolló un pipeline automatizado que:

1. Extrae datos de ventas desde archivos Excel
2. Limpia y transforma los datos con Python
3. Calcula métricas clave de negocio (KPIs)
4. Almacena los datos en una base de datos relacional (MySQL)
5. Visualiza la información en un dashboard interactivo en Power BI

---

## 🧠 Enfoque de Negocio

Este proyecto está diseñado con mentalidad de negocio, no solo técnica.

Se priorizaron indicadores clave como:

- Revenue (ingresos)
- Unidades vendidas
- Rendimiento por vendedor
- Performance por canal
- Tendencias temporales

El objetivo es responder preguntas como:

- ¿Qué productos generan más ingresos?
- ¿Qué vendedores tienen bajo rendimiento?
- ¿Qué canal es más rentable?
- ¿Cómo evolucionan las ventas en el tiempo?

---

## ⚙️ Tecnologías Utilizadas

- Python (ETL)
- Pandas (procesamiento de datos)
- MySQL (base de datos)
- SQL (consultas analíticas)
- Power BI (visualización)
- Excel (fuente de datos)

---

## 🔄 Pipeline de Datos


Excel → Python (ETL) → MySQL → Power BI Dashboard


---

## 📊 Dashboard (Power BI)

El dashboard fue diseñado para usuarios de negocio, con foco en claridad y toma de decisiones:

### KPIs principales:
- Total Revenue
- Total Unidades
- Ticket Promedio
- Ventas por Canal
- Rendimiento por Vendedor

### Visualizaciones:
- Evolución de ventas (línea temporal)
- Ventas por producto (barras)
- Ventas por canal (pie chart)
- Ranking de vendedores
